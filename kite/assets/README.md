# Images des spots

Déposez ici vos photos ou logos, avec exactement ces noms de fichiers :

| Fichier              | Spot                              |
| -------------------- | --------------------------------- |
| `le-jai.jpg`         | Le Jaï (Marignane)                |
| `carro.jpg`          | Carro (Martigues)                 |
| `pointe-rouge.jpg`   | Pointe Rouge (Marseille)          |
| `plage-napoleon.jpg` | Plage Napoléon (Port-Saint-Louis) |
| `la-ciotat.jpg`      | La Ciotat                         |

Format conseillé : JPEG ou WebP, ~800×500 px (l'image est recadrée en 140 px
de haut dans les cartes). Pour un autre nom ou format, modifiez le champ
`image` du spot dans `js/config.js`.

Tant qu'un fichier manque, le site affiche automatiquement un visuel de
remplacement avec le nom du spot.
